# TeamSparkOfRights

A modern, responsive website for TeamSparkOfRights built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Adapts seamlessly to different screen sizes and devices
- **Modern UI**: Clean, professional interface with a warm color scheme
- **Sticky Navigation**: Header remains accessible while scrolling
- **Interactive Elements**: Smooth hover effects and transitions

## Technology Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with gradients and animations
- **JavaScript**: jQuery library for enhanced interactivity
- **Responsive**: Mobile-first approach with viewport optimization

## Project Structure

```
TeamSparkOfRights/
├── index.html      # Main website entry point
└── README.md       # This file
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required (jQuery is loaded from CDN)

## Color Scheme

- **Primary Color**: #111827 (Dark Gray)
- **Accent Color**: #fbbf24 (Gold/Yellow)
- **Background**: #ffffff (White)
- **Text**: #6b7280 (Medium Gray for secondary text)

## Browser Compatibility

Works with all modern browsers that support:
- HTML5
- CSS3 (Flexbox, Gradients, Backdrop Filter)
- ES6 JavaScript
